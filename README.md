# GESTÃO SDI

Sistema de gerenciamento de escalas do Serviço de Diagnóstico por Imagem (SDI): cadastro de profissionais (RX, TC, RM e sobreaviso), geração e edição de escala mensal, escala de sobreaviso, feriados nacionais, verificação de descanso mínimo de 12h, relatórios e impressão.

Este projeto evoluiu do app "Sobreaviso Radiologia" (repositório `sobreaviso`), incorporando o módulo completo de gerenciamento de escalas.
